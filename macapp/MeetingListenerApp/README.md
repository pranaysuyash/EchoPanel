# MeetingListenerApp

## Build
```bash
cd macapp/MeetingListenerApp
swift build
```

## Run in Xcode
```bash
open Package.swift
```

## Keyboard shortcuts
- Cmd + Shift + L toggles listening
- Cmd + C copies Markdown from the panel

## Notes
- Requires macOS 13 or newer
- Screen recording permission is required for system audio capture
