# MeetingListenerApp (macOS 13+)

SwiftUI menu bar app with a floating side panel for:
- live transcript
- actions, decisions, risks
- entities

## Build
```sh
cd macapp/MeetingListenerApp
swift build
```

## Run (from Xcode recommended)
SwiftPM can build the executable, but to run a menu bar app reliably, open the package in Xcode and run `MeetingListenerApp`.
