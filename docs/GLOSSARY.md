# Glossary

- ASR: Automatic Speech Recognition.
- Partial: Interim transcript that may change.
- Final: Transcript segment considered stable and appended permanently.
- Cards: Structured artifacts extracted from transcript (Actions, Decisions, Risks).
- Entities: People, orgs, dates, projects, topics extracted from transcript.
- ScreenCaptureKit: Apple framework used to capture screen and system audio (permission gated).
- Screen Recording permission: macOS privacy permission required for ScreenCaptureKit system audio capture.
