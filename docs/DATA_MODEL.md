# Data Model (v0.1)

This document describes the in-app and exported data shapes. The transport format is specified in `docs/WS_CONTRACT.md`.

## Session
- `session_id`: UUID string generated by the client at Start and reused for the full session.
- `started_at`: local timestamp
- `ended_at`: local timestamp
- `source`: `system_audio` (v0.1 only)
- `sample_rate`: 16000
- `channels`: 1
- `format`: `pcm_s16le`

## Transcript
Transcript is append-only for finalized segments plus an optional active partial.

Segment:
- `t0`: seconds from session start (float)
- `t1`: seconds from session start (float)
- `text`: string
- `is_final`: boolean
- `confidence`: float 0..1

## Cards
### Action
- `text`: string
- `owner`: string or null
- `due`: `YYYY-MM-DD` or null
- `confidence`: float 0..1
- `evidence`: list of evidence items

### Decision
- `text`: string
- `confidence`: float 0..1
- `evidence`: list of evidence items

### Risk
- `text`: string
- `confidence`: float 0..1
- `evidence`: list of evidence items

Evidence:
- `t0`, `t1`: seconds
- `quote`: string

## Entities
Entity:
- `name`: string
- `type`: one of `person|org|date|project|topic` (v0.1 categories)
- `last_seen`: seconds
- `confidence`: float 0..1

## Final summary
- `markdown`: string
- `json`: object (backend-defined, but must include the same high-level groups as the UI: transcript, actions, decisions, risks, entities)
