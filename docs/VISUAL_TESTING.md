# Visual Testing (manual)

Automated visual tests are not set up for the macOS app in v0.1. Use this manual checklist.

## Side panel layout
- Three lanes are visible: Transcript, Cards, Entities.
- Transcript lane shows partial text in lighter style.
- Final segments appear with timestamps and confidence.
- Cards lane shows Actions, Decisions, Risks sections.
- Entities lane shows name, type, last seen, confidence.

## Menu bar behavior
- Timer increments while listening.
- Menu bar shows Listening vs Idle state.
- Cmd+Shift+L toggles Start/Stop from anywhere.
- Cmd+C copies Markdown from the panel.
