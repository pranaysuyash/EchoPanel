# Competitor Accounts Setup (Plan)

This document describes a non-production, offline plan to prepare accounts for hands-on competitive analysis.

Note: Do not create real paid accounts without explicit authorization. Use trial/alerts and internal testing data only.

Sections:
- Accounts to Create
- Data to Capture
- Access & Testing Plan
- Example Checklist
- Data Handling & Ethics

## Accounts to Create (Priority)
- Otter.ai (trial account)
- Fireflies.ai (trial account)
- Granola (trial or free tier)

## Data to Capture
- Account URL
- Sign-up date
- Plan used
- Test scenarios run
- Findings summary
- Tester's code/ID (keep this internal)
- Date of testing

## Access & Testing Plan
- For each account, sign in with test credentials
- Run Week 1 Scenarios 1-3 (Back-to-Back, 1-hour, Multi-Participant)
- Take screenshots and notes
- Record latency, accuracy, UX impressions
- Export options tested (Markdown/JSON)

## Data Handling & Ethics
- Use non-production accounts; do not collect sensitive data
- Remove any personal data after testing
- Store results in encrypted/local repository
