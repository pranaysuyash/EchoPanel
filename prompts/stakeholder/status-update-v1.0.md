# Status Update (EchoPanel) — v1.1

## When to use
Weekly update, release update, or stakeholder summary.

## Output
- Highlights (3–6 bullets)
- Shipped (with dates)
- In progress
- Risks / blockers
- Next up (1–2 weeks)
- Asks (if any)

## Constraints
- Use concrete dates (YYYY-MM-DD) for shipped items.
- If a claim depends on a run/test, include the command and outcome as Observed.
